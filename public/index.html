Image Search Abstraction API

User Stories

1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.

2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.

3. I can get a list of the most recently submitted search strings.

  • Instructions

  • Perform a search by accessing https://img-ninja.herokuapp.com/search/<query> with your query at the end.
    Adding ?offset=<page-number> will paginate the results, 1 being the default, 2 being the second page, etcetera.

  • Check out the 10 most recent searches by heading over to https://img-ninja.herokuapp.com/recent.
